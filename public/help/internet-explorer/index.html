Streetmix and Internet Explorer

Streetmix does not currently support any version of Internet Explorer because it lacks proper implementation of many modern web standards that are common in other major browsers, such as Chrome, Firefox or Safari. Learn more about upgrading your browser.

Will Streetmix ever support Internet Explorer?

We believe that Internet Explorer 10 and 11 are likely candidates for Streetmix support at some point in the future, but we don't yet have a timeline of when that would occur. We will not be able to support Internet Explorer 9 or lower while retaining any Streetmix functionality.

Does Streetmix require plugins?

Streetmix does not rely on any external plugins (like Java, Flash, Silverlight, etc). It uses only features of modern web standards such as HTML5. For example:

  • Canvas
  • CSS pointer-events

In addition, Internet Explorer has a lot of bugs, quirks, or holes in its handling of the modern HTML/CSS/JavaScript specification. As a result, even if we were able to work around all of these, it would require an immense amount of time and effort.
